# <刀剣log>

## サイト概要
### サイトテーマ
- 刀の鑑賞記録や、得た情報をまとめて共有できる「まとめサイト」

### テーマを選んだ理由
- 刀剣を鑑賞した際の感想や、撮影した写真を分類したり、<br>
情報やそのソースをまとめるためのツールが欲しいと思ったから

### ターゲットユーザ
- 刀剣の鑑賞をする人
- 刀剣に興味ある人

### 主な利用シーン
- 刀剣の展示を鑑賞した後
- 鑑賞前の予習
- 得た知識の整理
- 刀剣の来歴やそのソースをまとめる
